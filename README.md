# 🥁 Drum Kit Web Application

An interactive **Drum Kit Web App** built using **HTML, CSS, and JavaScript** that plays different drum sounds when users press keyboard keys or click on on-screen drum buttons.

This project is designed to practice **DOM manipulation, event handling, audio control, and interactive frontend development**.

---

## 🚀 Features

- 🎵 Play different drum sounds using keyboard keys  
- 🖱️ Click drum buttons with mouse input  
- ✨ Smooth button press animations  
- 🎨 Clean and responsive UI design  
- ⚡ Real-time sound triggering with JavaScript  

---

## 🛠 Tech Stack

| Technology | Usage |
|-----------|------|
| HTML5 | Structure of webpage |
| CSS3 | Styling and layout |
| JavaScript | Logic, events, sound control |

---

## 🎮 Controls

Use keyboard keys:

| Key | Sound |
|----|------|
| W | Tom 1 |
| A | Tom 2 |
| S | Tom 3 |
| D | Tom 4 |
| J | Snare |
| K | Crash |
| L | Kick Bass |

You can also click the buttons on screen.

How It Works
User presses a key or clicks a button
JavaScript detects the event using event listeners
Corresponding drum sound is played
Button animation is triggered for visual feedback
